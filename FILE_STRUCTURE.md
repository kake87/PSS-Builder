## Backend (Python/FastAPI)

### `backend/requirements.txt`
- Зависимости Python проекта
- FastAPI, Uvicorn, Pydantic и др.

### `backend/app/main.py`
- Главное приложение FastAPI
- Инициализация сервера
- Маршруты и middleware (CORS)

### `backend/app/config.py`
- Конфигурация приложения
- Параметры API, БД, функций
- Загрузка из .env файлов

### `backend/app/storage.py`
- Система хранения данных в памяти
- Класс InMemoryStorage с методами CRUD
- Встроенные шаблоны и правила

### `backend/app/equipment_catalog.py`
- Каталог реального оборудования
- 11 предустановленных моделей
- Камеры, коммутаторы, серверы, контроллеры, UPS

### Модели данных (`backend/app/models/`)

| Файл | Описание |
|------|----------|
| `__init__.py` | Экспорт всех моделей |
| `device.py` | Device, Port, DeviceType, PortType |
| `link.py` | Link, Cable, CableType |
| `rule.py` | Rule, RuleType, DEFAULT_RULES |
| `project.py` | Project с методами валидации |
| `template.py` | Template, DEFAULT_TEMPLATES |

### API маршруты (`backend/app/api/`)

| Файл | Endpoints |
|------|-----------|
| `__init__.py` | Объединение всех маршрутов |
| `projects.py` | GET/POST/DELETE проектов, валидация |
| `devices.py` | Управление устройствами, каталог |
| `templates.py` | Работа с шаблонами решений |
| `rules.py` | Получение и фильтрация правил |

### Schemas (`backend/app/schemas/`)

| Файл | Описание |
|------|----------|
| `__init__.py` | Pydantic схемы для API |

## Развёртывание

### Локально
```bash
python run.py
http://localhost:8000
```

### Docker (планируется)
```dockerfile
FROM python:3.11
WORKDIR /app
COPY backend/requirements.txt .
RUN pip install -r requirements.txt
COPY backend ./app
CMD ["python", "-m", "uvicorn", "app.main:app", "--host", "0.0.0.0"]
```
