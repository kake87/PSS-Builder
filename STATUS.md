## 🚀 Быстрый старт

### 1. Установка и запуск Backend

```bash
# Перейти в директорию
cd "c:\Users\user\Desktop\Python\Codes\PSS builder"

# Установить зависимости
pip install -r backend/requirements.txt

# Запустить приложение
python run.py

# Приложение доступно по адресу:
http://localhost:8000

# API документация:
http://localhost:8000/docs
```

### 2. Запуск примеров

```bash
python example.py
```

Результат:
- ✅ Малый офис с 3 камерами (протестировано)
- ✅ Средний склад с 5 камерами (протестировано)
- ✅ Валидация проектов
- ✅ Вывод информации о системе

### 3. Использование API

```bash
# Создать проект
curl -X POST http://localhost:8000/api/projects \
  -d '{"name":"My System","description":"Test"}'

# Получить проекты
curl http://localhost:8000/api/projects

# Получить каталог оборудования
curl http://localhost:8000/api/equipment-catalog

# Получить шаблоны
curl http://localhost:8000/api/templates

# Получить правила
curl http://localhost:8000/api/rules
```

## 🎯 Основные возможности

### 1. Управление проектами
- Создание, редактирование, удаление проектов
- Сохранение конфигурации плана архитектуры
- История изменений (планируется)

### 2. Картлог оборудования
- 11+ предустановленных моделей оборудования
- Полные спецификации (порты, питание, бандвидт)
- Легко добавляемые новые модели

### 3. Построение архитектуры
- Добавление устройств в проект
- Создание соединений между портами
- Выбор типов кабелей и расстояний
- Визуализация графа (планируется)

### 4. Автоматическая валидация
- Проверка целостности соединений
- Проверка типов кабелей
- Расчёт требований к питанию
- Проверка требований к пропускной способности
- Проверка требований к хранилищу

### 5. Готовые шаблоны решений
- Малый офис (4-6 камер)
- Средний склад (15-20 камер)
- Крупный магазин (40+ камер)
- Производство (60+ камер)

### 6. API-first дизайн
- Полный REST API
- Swagger UI документация
- Возможность интеграции с другими системами

## 📝 Примеры API запросов

### Создание проекта
```bash
POST /api/projects
{
  "name": "Warehouse Project",
  "description": "Main facility security"
}
```

### Добавление устройства из каталога
```bash
POST /api/projects/{project_id}/devices-from-template
?equipment_key=hikvision-4k
```

### Получение всех доступных правил
```bash
GET /api/rules
```

### Валидация проекта
```bash
POST /api/projects/{project_id}/validate
```
